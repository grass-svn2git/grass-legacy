NAME

d.mon.viewproj- Associates the view projection with the graphics monitor.
(GRASS Viewproj Display Program)

SYNOPSIS

d.mon.viewproj
d.mon.viewproj help

DESCRIPTION

d.mon.viewproj  associates the current view projection (set by d.auto.viewproj or d.set.viewproj) with the current monitor.

The projections are based upon PROJ (originally from the USGS).    More information about PROJ is available at  PROJ.4- Cartographic Projections Library

Viewproj programs work only with  lat-lon data.
