NAME

d.auto.viewproj- Automatically sets the current view projection and adjusts the current region.
(GRASS Viewproj Display Program)

SYNOPSIS

d.auto.viewproj
d.auto.viewproj help
d.auto.viewproj [-r] projname=projection

DESCRIPTION

d.auto.viewproj automatically adjusts the current region based upon the view projection name (projname=projection), calculates PROJ parameters based upon the region, and sets the current view projection.  It is used in place of d.set.viewproj and is useful for the user who is unfamiliar with PROJ.

The projections are based upon PROJ (originally from the USGS).    More information about PROJ is available at  PROJ.4- Cartographic Projections Library

Viewproj programs work only with  lat-lon data.

Flags:

-r
Retain the current region

Parameters:

projname=projection
Name of a projection
Run d.auto.viewproj help for a list of the projection names.  If the user does not use any arguments, the program will prompt the user to enter them.
