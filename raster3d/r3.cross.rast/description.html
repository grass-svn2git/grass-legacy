DESCRIPTION

This module creates a cross section 2d map from one g3d raster map based on a 2d elevation map. It checks whether the value of the elevation map is located in the z-coordinate of the 3d map. In that case the 3d map value for this position is copied to the cross section output map, otherwise the NULL value is set.

ATTENTION

The region settings for 2d raster and 3d raster maps have to be equal (res == res3)!

NOTES

To create a cut plane elevation map use r.mapcalc. Some examples:
  • To create a cut plane elevation map in x direction type
    r.mapcalc "cutplane = col()*x"
    , x is the value for your elevation. If the range of col() is 1 ... 10, the elevation map has the range 1 ... 10 if x == 1 and if x == 10 the range 10 ... 100
  • To create a cut plane elevation map in y direction type
    r.mapcalc "cutplane = row()*x"
    , x is the value for your elevation. If the range of col() is 1 ... 10, the elevation map has the range 1 ... 10 if x == 1 and if x == 10 the range 10 ... 100
  • You can also make a cut in y and x direction with r.mapcalc, just type
    r.mapcalc "cutplane = (row()+col())*x"

EXAMPLE

Simple Spearfish example

g.region -d
g.region res=150 res3=150 t=1000 b=0 tbres=100

# synthetic data, could be geological structures:
r3.mapcalc "map3d=sin(row())+sin(col())+sin(depth()*depth())"

#create a cutplane map
r.mapcalc "cutplane = col()*10"

#create the cross section map
r3.cross.rast input=map3d elevation=cutplane output=crosssection
