DESCRIPTION

db.createdb allows the user to create a new empty database through different drivers. A working database connection needs to be established.

EXAMPLE

Create a new PostgreSQL database (if PostgreSQL connection is established through odbc driver)
db.createdb driver=odbc database=grass60test

Create a new PostgreSQL database (if PostgreSQL connection is established through pg driver)

db.createdb driver=pg database='host=pgserver.itc.it,dbname=grass60test'
