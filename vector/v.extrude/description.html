DESCRIPTION

v.extrude creates 3D faces or lines based on 2D vector objects, i.e. points become 3D lines; lines, boundaries or areas become faces.

If the flag -t is used then faces ("walls" and "roofs") follow the elevation model by using individual elevation values for the vertices and nodes. This can be useful for models of large objects (forest stands).

EXAMPLES

3D houses with fixed height

v.extrude input=houses output=houses3D height=5

3D houses with individual height

v.extrude input=houses output=houses3D elevation=dem hcolumn=height
