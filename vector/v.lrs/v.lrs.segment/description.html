DESCRIPTION

v.lrs.segment creates points/segments from input lines, linear reference system and positions read from standard in or a file. The format is as follows:
P <point_id> <line_id> <milepost>+<offset> [<side offset>]
L <segment_id> <line_id> <milepost>+<offset> <milepost>+<offset> [<side offset>]
