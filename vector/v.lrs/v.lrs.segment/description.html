DESCRIPTION

v.lrs.segment creates points/segments from input lines, linear reference system and positions read from standard in format, with pid: point id; sid: segment id; lid: line id: 
P <pid> <lid> <milepost>+<offset> [<side offset>]
L <sid> <lid> <milepost>+<offset> <milepost>+<offset> [<side offset>]
