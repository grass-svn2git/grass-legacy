DESCRIPTION

s.kernel generates a raster density map from point data using a moving 2D isotropic Gaussian kernel.

NOTES

s.kernel uses moving 2D isotropic Gaussian kernel.

LIMITATIONS

The modules only considers the presence of point, but not (yet) any attribute values.
