DESCRIPTION

v.convert converts between GRASS vector versions 5.0/5.3 and 5.7.

NOTES

Vector maps from 5.0/5.3 and 5.7 do not interfere. They are stored in different directories, so you can use the same names.

EXAMPLE

v.convert in=vectormap_from_50 out=vectormap_57

NOTE

If needed, to convert back from 5.7 into 5.0/5.3 vector format, use v.out.ogr (to SHAPE format) and then v.in.shape of the old GRASS program.
