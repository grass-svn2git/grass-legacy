DESCRIPTION

v.format is an interactive GUI (Graphical User Interface) which enables the user to register external SHAPE file into GRASS and allows to define PostGRASS connection.

FUNCTIONS

1) External format (SHAPEfile)

Vector name

Name of GRASS vector output.

SHAPEfile

Name of SHAPEfile to register (this does not import it).

Header informations [optional]

Here you can store some basic meta data comming with the SHAPEfile.

Organization = Name of organization you work for (GRASS Team)
Digit date = Date when map was digitized (7/04/92)
Digit name = Name of digitizer (GRASSuser)
Map name = Name of the map (testmap)
Map date = Date of import (todays date)
Map scale = Scale of map (1:24000)
Other info = Further useful information

Create vector

Creates GRASS vector file from SHAPEfile.

2) Output format (Native or PostGIS)

Output format

Can be native (GRASS internal) or PostGIS.

PostGIS database

Connection string to connect to PostGIS Server:

Example: host=myserver.itc.it,dbname=mydb,user=name

If 'password' is not specified (warning: clear text!), then the user will be session-wise queried for the password upon data queries.

Close

Close v.format GUI.

NOTES

To really import SHAPE vector files, use v.in.ogr instead.
