DESCRIPTION

v.format is an interactive GUI (Graphical User Interface) which enables the user to import external shapefile into GRASS vector format.

FUNCTIONS

Vector name

Name of GRASS vector output.

Shapefile

Name of shapefile to import.

Header informations [optional]

Here you can store some basic meta data comming with the shapefile.

Organization = Name of organization you work for (GRASS Team)
Digit date = Date when map was digitized (7/04/92)
Digit name = Name of digitizer (GRASSuser)
Map name = Name of the map (testmap)
Map date = Date of import (todays date)
Map scale = Scale of map (1:24000)
Other info = Further useful information

Create vector

Creates GRASS vector file from shapefile.

Close

close v.format GUI.
