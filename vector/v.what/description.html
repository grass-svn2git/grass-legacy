NAME

v.what - Allows user to query a vector layer by supplying a set of coordinates. Used by gis.m to provide interactive query feature.

SYNOPSIS

v.what
v.what help
v.what [-da] [east_north=east,north][distance=value] [map=string[,string,...]]

Flags:

-d
Print topological information (debugging).
-a
Attributes. Prints object attribute values.

Parameters:

east_north=east,north
Easting and northing values for location to be queried.
distance=value
Maximum distance from point coordinates to look for an object.
map=string
Name of existing vector map.

DESCRIPTION

v.what Outputs the category value(s) associated with user-specified location(s) in user-specified vector map layer(s). This module was derived from the d.what.vect module by removing all interactive code and modification of the output for easy parsing.
