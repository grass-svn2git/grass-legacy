DESCRIPTION

v.distance finds for features from one vector (from=) nearest feature in another vector (to=). Various informations about reation (distance, ...) may be uploaded to attribute table attached to first vector or printed to stdout. New vector may be created where line connecting nearest points on features are written.

EXAMPLE

Find nearest lines in vector ln for points from vector pnt and write line categories to column linecat in attribute table attached to pnt:
v.distance from=pnt to=ln upload=cat column=linecat

For each point from vector pnt find area from ar it is within and write area category to column areacat in attribute table attached to pnt:
v.distance from=pnt to=ar upload=cat column=areacat

NOTES

If nearest feature does not have a category, column is updated to null. This is true also for areas, which means for example, that if point is in island (area WITHOUT category), v.distance does not search for nearest area WITH category, island is identified as nearest and category updated to null.
