DESCRIPTION

v.drape converts 2D vector point or line data into 3D vector format via sampling of an elevation surface. Three sampling algorithms adapted from v.sample were incorporated into this module: nearest neighbor, bilinear, and cubic convultion.

NOTES

Please run v.region vect=2D_vector, and make sure that the extent of the elevation raster is at least as big as the vector to convert.

Additional vertices can be added to the input 2D vector with v.split.

Can be used in conjunction with v.out.pov and r.out.pov to export a complete set of vector and raster data for display in POVRAY.

EXAMPLE

v.drape [-bc] input=2D_vector [type=line[,point,...]] rast=elevation_surface output=3d_vector

POVRAY EXAMPLE

#setup the region
g.region rast=elevation
#export the vector data
v.drape in=2d_roads out=3droads raster=elevation
v.out.pov 3droads out=3droads.pov
#export the raster data
r.out.pov elevation elevation.tga
r.out.png landsat landsat.png
