DESCRIPTION

v.in.dwg imports DWG/DXF file into GRASS.

EXAMPLE

v.in.dwg input=map.dwg output=map

NOTES

v.in.dwg requires OpenDWG toolkit. To get this toolkit you must become at least "Associate Member" of OpenDWG Alliance (http://www.opendwg.org), it is free-of-charge (you have to fill one form).

Then unpack library and copy to this directory:
ad2.a ad2.h odio.h adinit.dat
and compile this module.

Not all entity types are supported (warning printed).
