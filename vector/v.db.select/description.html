DESCRIPTION

v.db.select prints attributes of a vector map.

EXAMPLE

v.db.select roads
cat|label
0|no data
1|interstate
2|primary highway, hard surface
3|secondary highway, hard surface
4|light-duty road, improved surface
5|unimproved road
