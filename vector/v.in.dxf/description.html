DESCRIPTION

Standard DXF is imported. The following graphical objects are supported:
  • GRASS point type:
    • DXF POINT
  • GRASS line type:
    • DXF LINE
    • DXF POLYLINE
    • DXF LWPOLYLINE
    • DXF ARC
    • DXF TEXT
  • GRASS face type:
    • DXF MESHES
