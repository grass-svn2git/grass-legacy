DESCRIPTION

v.category attaches, deletes or reports vector categories.

EXAMPLES

report vector categories
v.category in=testmap option=report 
FIELD 1:
type       count        min        max
point          0          0          0
line        1379          1       1379
boundary       0          0          0
centroid       0          0          0
area           0          0          0
all         1379          1       1379
delete all vector categories in field 1
v.category in=testmap out=outmap option=del

add vector categories in field 1 with step=2
v.category in=outmap out=stepmap option=add step=2 

FIELD 1:
type       count        min        max

line        1379          1       2757
boundary       0          0          0
centroid       0          0          0
area           0          0          0

Results can be tested using d.what.vect.
