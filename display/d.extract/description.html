DESCRIPTION

d.extract allows a user to graphically select vector objects from an existing vector map and creates a new map containing only the selected objects. The database table is copied over to the new map.

BUGS

The user selection box may select too many vectors. A test is needed if vector line really intersects selection box, and not only vector outline box intersects selection box.
