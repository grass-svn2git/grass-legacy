DESCRIPTION

This program allows user to start selected graphical user interface (GUI) from command line prompt.

Currently three different GUIs are supported:

  • oldtcltk - Old TCL/TK based GUI aka Display Manager
  • tcltk - Current TCL/TK based GUI aka GIS Manager
  • wxpython - New wxPython based GUI aka wxGUI
If GRASS_GUI variable is not set or differs from selected option then GRASS_GUI is changed, selected GUI is used as default UI.
