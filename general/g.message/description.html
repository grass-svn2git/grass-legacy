DESCRIPTION

This program is to be used in shell/perl/python scripts, so the author does not need to use echo program. The advantage of g.message is that it formats messages just like other GRASS modules do and that its functionality is influenced by the GRASS_VERBOSE environment variable.

The program can be used for standard informative messages as well as warnings (-w flag) and fatal errors (-e flag). For debugging purposes, the -d flag will cause g.message to print level 1 debugging message.

NOTES

Messages containing "=" must use the message= syntax so the parser doesn't get confused.

Messges containing '!' and '$' must use single 'quotes' to avoid unwanted shell translation.

If you want a long message (mutli-line) to be dealt with as a single paragraph, use a single call to g.message with text split in the script using the backslash as the last character. (In shell scripts don't close the "quote")

A blank line may be obtained with: g.message ""

Redundant whitespace will be stripped away.
