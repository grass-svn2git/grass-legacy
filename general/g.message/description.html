DESCRIPTION

This program is to be used in shell/perl/python scripts, so the author does not need to use echo program. Advantage of g.message is, that it formats messages just like other GRASS modules do and that it's functionality is influenced by GRASS_VERBOSE environment variable.

The program can be used for standard informative messages as well as warnings (flag -w) and fatal errors (flag -e). For debugging purposes, flag -d will cause g.message to print level 1 debugging message.
