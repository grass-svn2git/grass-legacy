NAME

nviz - nviz - Visualization and animation tool for GRASS data

SYNOPSIS

nviz
nviz help
nviz [-qkxv] [elevation=string[,string,...]] [color=string[,string,...]] [vector=string[,string,...]] [points=string[,string,...]] [volume=string[,string,...]] [path=string] [script=string] [state=string]

Flags:

-q
Quickstart - Do not load any data
-k
Script kill option
-x
Start in Demo mode
-v
Output more comments (default=quiet)

Parameters:

elevation=string[,string,...]
Raster file(s) for Elevation
color=string[,string,...]
Raster file(s) for Color
vector=string[,string,...]
Vector lines/areas overlay file(s)
points=string[,string,...]
Vector points overlay file(s)
volume=string[,string,...]
Name of existing 3dcell map
path=string
Set alternative panel path
script=string
Execute script file at startup
state=string
Load previosly saved state file

DESCRIPTION

nviz - nviz - Visualization and animation tool for GRASS data. Please refer to the detailed manual inside NVIZ or here.
