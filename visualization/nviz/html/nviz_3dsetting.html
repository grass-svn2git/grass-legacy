3D Settings

You can save or load a particular 3D view. NVIZ saves 3D view files in the GRASS 3d.view directory. The 3D view files include the current view in real coordinates, lighting options, mesh resolution, and surface resolution. These files use the same format as the GRASS program d.3d.

Load 3d Settings

Loads a previously saved 3D view. The Movement panel updates to the new view.

Save 3d Settings

Saves the current 3D view to a user defined file.
