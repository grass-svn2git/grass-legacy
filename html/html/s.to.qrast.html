NAME

s.to.qrast

SYNOPSIS

s.to.qrast
s.to.qrast help
s.to.qrastinput=string rast=string [type=string] [findex=integer]

DESCRIPTION

s.to.qrast converts a GRASS site file to a GRASS raster file allowing to compute for each cell the minimum, maximum, mean, sqm of the site points belonging to it.

Parameters:

input=string
input site list file name
rast=string
output raster file name
type=string
computed statistic
Options: min,max,mean,sqm
Default: mean
findex=integer
Attribute field number to use for operations
Default: 0
