NAME

d.info - Display information about the active display monitor

SYNOPSIS

d.info
d.info help
d.info [-rdc]

Flags:

-r
Display screen rectangle
-d
Display screen dimensions
-c
Display number of colors

DESCRIPTION

d.info displays information about the active display monitor
