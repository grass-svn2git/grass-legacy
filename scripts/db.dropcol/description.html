DESCRIPTION

db.dropcol drops a column from an attribute table.

NOTES

db.dropcol is a front-end to db.execute to allow easier usage with a special workaround for the SQLite driver to support column drop.

EXAMPLES

Dropping a column:
db.dropcol sentiero_brenta_points column=slope
db.describe -c sentiero_brenta_points

SEE ALSO

db.execute, v.db.dropcol
