DESCRIPTION

v.db.univar - Calculates basic univariate statistics for numeric attributes in a data table. It will calculate minimum, maximum, range, mean, standard deviation, variance, coefficient of variation, quartiles, median, and 90th percentile. It uses db.select to create list values for statistical calculations. (GRASS Shell Script)

NOTES

Unless the database and driver are specified, it default the values set in db.connect.

EXAMPLE

g.region rast=elevation.10m -p
v.random out=samples n=100
v.db.addtable samples col="heights double precision"
v.what.rast samples rast=elevation.10m col=heights
v.db.select samples

v.db.univar samples col=heights
