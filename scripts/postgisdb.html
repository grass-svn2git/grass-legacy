DESCRIPTION

postgisdb is used to create and destroy PostGrass databases. All options that are not available in primary options scope will be ignored. Only Postmaster can run this script to create or delete a new database (use -U to define database owner)

EXAMPLE

this example from the GRASS 5.1 tutorial pages needs the file postgis.sql in the current directory. 

postgisdb -c botswanaUTM34 -h pgserver.itc.it -U postgres -G ./
