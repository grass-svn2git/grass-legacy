DESCRIPTION

v.univar.sh - Calculates basic univariate statistics for numeric attributes in a data table. It will calculate minimum, maximum, range, mean, standard deviation, variance, coefficient of variation, quartiles, median, and 90th percentile. It is closely based on r.univar.sh and uses db.select to create a list values for statistical calculations. This script could be replaced by an updated v.univar.
(GRASS Shell Script)

NOTES

Unless the database and driver are specified, it default the values set in db.connect.
