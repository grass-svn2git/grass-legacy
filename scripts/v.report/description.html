DESCRIPTION

generates a table showing the area present in each of the categories of a user-selected data layer. Area is given in hectares, square meters, and square kilometers. If the units option is used, area is given in acres, square feet, and square miles. v.report works on the full map data; therefore, the current region is ignored. If statistics shall be spatially limited, a map subset has to be created with v.in.region and v.overlay and then v.report is run on the new map.

EXAMPLE

Spearfish example: 
v.report fields option=area units=hectares

SEE ALSO

v.in.region, v.overlay
