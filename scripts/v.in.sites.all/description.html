DESCRIPTION

v.in.sites.all converts all GRASS 5.0/5.3/5.4 sites files in the current mapset into 5.7 vectors.

NOTES

Sites maps from 5.0/5.3/5.4 and vector maps from 5.7 do not interfere. They are stored in different directories, so you can use the same names. Old sites maps can be listed with g.list sites. As this GRASS version uses SQL for attribute management, there are some SQL restrictings concerning the file names. This script changes dots (e.g. "foo.bar") in old vector map names into underline(s) (e.g. "foo_bar").

EXAMPLE

To convert all old vector maps in the current mapset to the new vector format:
v.in.sites.all -r
