DESCRIPTION

r.what.color outputs the color associated with user-specified category values in a raster input map. Values may be specified either using the values= option, or by specifying the -i flag and passing the values on stdin, one per line. For each value which is specified, a line of output will be generated consisting of the category value followed by the color in RR:GG:BB hexadecimal notation, e.g.: 
1500.000000: 00:33:ff
If the input map is an integer (CELL) map, the category will be written as an integer (no decimal point), otherwise it will be written in floating point format (printf("%f") format). If the lookup fails for a value, no output will be generated for that value. Thus, when using the module from scripts or other programs, it is important to identify output lines by the actual value, rather than simply assuming one line for each value.
