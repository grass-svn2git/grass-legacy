DESCRIPTION

r.li.patchnumber is a patch index for r.li program.
It calculates the function
f(sample_area)= Patch_Number
This index is calculated using a 4 neighbour algorithm

EXAMPLES

To calculate patch number index on map my_map, using ~/.r.li/history/my_conf configuration file and saving results in my_out file run: 
r.li.patchnumber map=my_map conf=my_conf out=my_out
Note that you MUST NOT USE ABSOLUTE PATH NAME for output files, because conf output files are raster if moving window disposition was selected, otherwise they are in ~/.r.li/output folder.

NOTES

Do not use absolute path names for output files. They are raster if moving window disposition was selected, otherwise they are in ~/.r.li/output folder. An that is full of null values is considered to have zero patch.
If you want to have null values instead run
r.null setnull=0 map=my_map
after index calculation.
