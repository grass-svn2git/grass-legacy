DESCRIPTION

r.out.gdal allows a user to export a GRASS raster map layer into any GDAL supported raster file format.

GDAL supported raster formats

Full details on GDAL supported formats are available at:

http://www.gdal.org/formats_list.html

EXAMPLES

r.out.gdal2 input=roads output=roads.tif format=GTiff

SEE ALSO

r.out.ascii, r.out.arc, r.out.bin
